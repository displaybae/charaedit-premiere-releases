# CharaEdit for Premiere Pro

영상 편집자를 위한 Premiere Pro 확장 — STT · AI 에이전트 · 자동 컷.

> 이 저장소는 **다운로드 배포용**입니다. 설치 파일은 [Releases](../../releases/latest)에서 받으세요. (소스는 비공개)

## 설치 (Windows 베타)

1. [최신 릴리즈](../../releases/latest)에서 **`CharaEdit-Windows.zip`** 다운로드
2. 압축 풀기 (바탕화면 등 아무데나)
3. 폴더 안 **`install.bat` 더블클릭** → 검은 창 끝나면 아무 키
4. **Premiere Pro 완전히 껐다 재시작**
5. 상단 메뉴 **Window ▸ Extensions ▸ CharaEdit**

### 안 보이거나 보안 경고가 뜨면
- "Windows가 PC를 보호했습니다" → **추가 정보 ▸ 실행** (인터넷서 받은 파일이라 그래요)
- 또는 `install.bat` 우클릭 ▸ 속성 ▸ **차단 해제** 체크 후 재실행

## 사용
1. 패널에서 **회원가입** (무료 크레딧 900)
2. 타임라인 **V1에 영상 클립** 올리고 선택
3. 편집 의도 / 길이 / 카드 장수 설정 → **분석 시작**
4. AI 채팅으로 컷 요청 → 검수 → **적용**

요구사항: Windows + Premiere Pro 2020 이상. ffmpeg 별도 설치 불필요(확장에 내장).
